Projects

Ruqi

National School Attendance Monitoring System

Built for the Syrian Ministry of Education. A multi-portal web app that connects the four tiers of a national school system — teacher → school → directorate → ministry (plus parent and admin portals) — around one shared source of truth for daily attendance and field reports. Backed by Supabase with extensive PostgreSQL Row-Level Security, automated with GitHub Actions CI/CD, and built Arabic-first with a full RTL UI.

  • Vanilla JS
  • Supabase
  • PostgreSQL (RLS)
  • GitHub Pages
  • Arabic RTL UI

UniManager

University Student Manager

An offline-first PWA for university students: weekly schedule, a multi-system GPA engine, exams, attendance, notes, tasks and study tracking — plus real-time collaborative groups with shared schedules and chat. Built on Supabase (Auth, PostgreSQL with RLS, Realtime, Storage) with an IndexedDB sync queue and a service worker for true offline support, and a bilingual Arabic/English UI with full RTL.

  • PWA
  • Supabase
  • Service Worker
  • Vanilla JS
  • Cloudflare Pages
